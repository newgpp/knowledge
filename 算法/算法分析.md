# 算法分析

一个程序运行的总时间一般跟两个因素有关:

- 执行每条语句的耗时
- 执行每条语句的频率

## 一、典型的近似

    private static int count(int[] a) {
        int n = a.length;
        int cnt = 0;
        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                for (int k = j + 1; k < n; k++) {
                    if (a[i] + a[j] + a[k] == 0) {
                        cnt++;
                    }
                }
            }
        }
        return cnt;
    }
